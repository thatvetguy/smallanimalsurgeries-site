# Small Animal Surgeries, Inc — Website

Static single-page website hosted on GitHub Pages at [smallanimalsurgeries.com](https://smallanimalsurgeries.com).

## Deployment

1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Set the source to the `main` branch (root `/`).
4. The `CNAME` file is already included for the custom domain. Ensure your DNS has a CNAME record pointing `smallanimalsurgeries.com` to `<your-github-username>.github.io`.
5. The site will be live at https://smallanimalsurgeries.com once DNS propagates.
